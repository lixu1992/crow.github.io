他们说1

2017-03-02

回到旧房子的时候已经是下午了，房子里的阳光没有上午那么耀人眼了，可依旧能把房间里照的亮堂。这恰到好处的光线让旧房子里的物件更显得多情了，有明有暗。也正因如此，也让他觉得忧伤了，也许是历历在目，也许是显得过于真切了。
透着窗户的亮光，可以明显的看的窗台上一层厚厚的灰尘。他没有心思去弹掉了。墙上挂的还是当初离开时的旧画，在如今看来，这些快要老掉牙的东西，却依旧让人亲切。
挂在墙上的时钟不知是什么时候停止的。时间停在了一点十二分的地方一动不动。他怔怔地盯着时钟看了好久，心中开始难过起来了。
时钟啊，你要是真的是停止了，也不至于如此地让他难过，毕竟你记录了他的所有欢乐时光。
可你只是停下了，用别的东西计量了。
他悲伤，可不想弹去那厚厚的灰尘。
